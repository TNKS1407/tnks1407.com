実機でやる ── MLX90640 + Raspberry Pi Pico

Picoは生データの土管、温度較正はこのページのJSがやる。ファームは firmware.zip（mlx_stream.py + README）。
MLX90640ボードPico物理ピンGPIO
VIN36 (3V3 OUT)-
GND38-
SDA6GP4
SCL7GP5
給電は3.3V厳守（5V/VBUS直結は禁止。素子は3.3Vデバイス）。
手順: ① pip install mpremote ② python -m mpremote connect COMx run mlx_stream.py で流れ出すのを確認 （COMxはデバイスマネージャか python -m mpremote connect list）③ mpremoteを Ctrl+Cで止めてから このページの「接続」でCOMを選ぶ （シリアルは1プロセス占有。runを止めてもスクリプトはPico上で走り続ける）。 常用は cp mlx_stream.py :main.py でUSB挿すだけ。
電源投入から数分は素子の自己発熱で絵が少しずつドリフトする。絶対値を見たい時は5分置いてから。
