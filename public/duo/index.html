実機でやる ── VL53L5CX + MLX90640 + Pico(1本のFWで両方)

ファームは firmware.zip （duo_stream.py + vl53l5cxドライバ(MIT, Mark Grosen氏) + README）。
信号Pico物理ピンGPIO備考
3V336-両センサ並列。3.3V厳守
GND38-両センサ並列
SDA6GP4両センサ並列(0x29/0x33で共存)
SCL7GP5両センサ並列
ToF LPn9GP6ToFのみ
ToFの給電線は太く短く直結。細いジャンパの継ぎ足しは電圧降下で初期化に失敗する(実測0.5V降下の前科)。
手順: ① zipを解凍 ② ドライバをPicoへ: python -m mpremote connect COMx mkdir :lib → mkdir :lib/vl53l5cx → vl53l5cx/ の4ファイルを cp ... :lib/vl53l5cx/（READMEにコピペ用コマンド）③ python -m mpremote connect COMx run duo_stream.py（ToF初期化~5秒）④ mpremoteをCtrl+Cで止めてから「接続」。単独ビューア(/tof/, /thermal/)も同じストリームに繋げる。
I2Cは400kHz＋timeout=500ms(実測: 1MHzはRP2040で不安定、既定timeout 50msはToFファーム転送で切れる)。シリアルは1プロセス占有。 COMはデバイスマネージャか python -m mpremote connect list。
